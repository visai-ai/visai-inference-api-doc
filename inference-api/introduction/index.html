Optical Character Recognition

OCR or Optical Image Recognition is the process of extracting a textual information from an image. In other word, it's an image to text stored the result as a series of character or text that is encoded by a specific character encoding method such as UTF-8. Thus, this makes it easier and more accurate for a computer to perform some tasks on these texts. For example, conducting document search from PDF files using OCR and information retrieval.
