List buckets

A GET request sent to the endpoint root returns a list of buckets that are associated with the specified service instance. For more information about endpoints, see Endpoints and storage locations.

Query Parameters
extended boolean

Provides LocationConstraint metadata in the listing.

Header Parameters
ibm-service-instance-id string — REQUIRED

List buckets that were created in this service instance.

Responses
200

ok
